Sistem **Pelayanan TIK** ini dibuat untuk mendukung proses pelayanan teknologi informasi di **Dinas Komunikasi dan Informatika (Diskominsa) Aceh**. Sistem ini dirancang agar admin, teknisi, dan tamu (guest) dapat berinteraksi sesuai peran masing-masing.

---

## ğŸš€ Fitur Utama

### 1. Dashboard Admin
- Memantau laporan masuk dan status penanganannya
- CRUD pengguna, teknisi, kategori layanan, dan penanggung jawab (penjab)

### 2. Manajemen Laporan
- Guest dapat membuat laporan
- Admin dapat meninjau, mengedit, dan memperbarui status laporan
- Teknisian dapat mengupdate progres laporan

### 3. Manajemen Pengguna
- Admin dapat menambah, mengedit, dan menghapus akun pengguna serta teknisi
- Guest dapat mendaftar, login, dan melihat laporan mereka sendiri

### 4. Kategori Layanan
- Admin dapat menambahkan kategori layanan baru
- Mengelompokkan laporan berdasarkan jenis layanan

### 5. Persetujuan Guest
- Admin dapat menyetujui atau menolak pendaftaran guest
- Mengelola akses tamu sesuai status persetujuan

### 6. Dokumen & KTP
- Menyimpan dan menampilkan dokumen dan KTP terkait laporan
- Mengakses file secara aman melalui sistem

### 7. Export Laporan
- Admin dapat mengekspor laporan ke format XLSX untuk keperluan administrasi

### 8. Download Template
- Guest dapat mengunduh template formulir untuk pengajuan laporan
